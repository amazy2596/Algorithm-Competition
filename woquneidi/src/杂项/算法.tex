\subsection*{树的重心}

定义（等价）：

\begin{enumerate}
    \item 最大子树最小
    \item 任意子树节点数 $\le \frac{n}{2}$
    \item 所有点到该点的距离和最小
\end{enumerate}

性质：

\begin{itemize}
    \item 最多两个重心，且相邻
    \item 加删叶子时重心最多移动一条边
    \item 合并两棵树，新重心在原重心路径上
    \item 正权树上最小距离和点为重心
\end{itemize}

\subsection*{树的直径}

求法：

\begin{enumerate}
    \item 两次 DFS（正权）

\end{enumerate}

性质：

\begin{itemize}
    \item 多条直径有共同公共部分
    \item 对任意节点，与其最远点集合中包含直径端点之一
\end{itemize}

\subsection*{生成函数}

普通生成函数：

\[
F(x)=\sum_{i\ge0} a_i x^i
\]

指数生成函数：

\[
F(x)=\sum_{i\ge0} a_i \frac{x^i}{i!}
\]

分别用于多重集的组合数、排列数问题。
